Zero-touch applicant onboarding system. Forward a candidate's email, and the pipeline auto-provisions a fully configured website with pre-installed templates, plugins, and licenses, then sends the applicant their credentials and test assignment.

0 Manual Steps

Pipeline
How It Works

One email triggers the entire onboarding flow. No dashboards, no manual steps.

1
Email Forwarded
A team member forwards an applicant's email to a dedicated inbox. The system monitors via IMAP and picks it up automatically.
IMAP Trigger n8n
2
Applicant Data Extraction
Custom JavaScript parses the forwarded email chain to extract the original sender's name and email. Handles multi-level forwards, Yahoo/GMX formatting, umlaut names, and HTML-encoded addresses.
JavaScript Regex Parsing
3
Subdomain & Credentials Generated
Applicant name is sanitized into a unique subdomain (umlaut handling, reserved name detection, collision avoidance). A strong 14-character password is generated.
Sanitization Crypto
4
Website Created
A PHP API on the server provisions a full CMS installation: subdomain DNS, database, CMS core, admin user, plus pre-installed page builder and form plugins with activated licenses.
PHP API CMS Page Builder Form Plugin
5
Logged to SharePoint
On success, the applicant's details, subdomain, URL, username, password, and database name are logged to a SharePoint list for team visibility. Failures go to a separate error list.
SharePoint API Error Handling
6
Onboarding Email Sent
The applicant receives a professional email with their WP admin URL, login credentials, the design brief, SharePoint link to brand assets, and the deadline for their test assignment. All fully templated.
SMTP HTML Email
